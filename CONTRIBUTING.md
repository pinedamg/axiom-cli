## How to Contribute

1. **Fork the Repo:** Create your own branch for any feature or fix.
2. **Follow the Style:** Adhere to the Rust coding standards and patterns established in the project.
3. **Test Your Changes:** Ensure that all tests pass before submitting a Pull Request.
4. **Submit a PR:** Provide a clear description of the changes and the problem they solve.

## Reporting Issues

Please use the GitHub Issue tracker to report bugs or suggest new features. For security-related issues, please contact the maintainers privately.
